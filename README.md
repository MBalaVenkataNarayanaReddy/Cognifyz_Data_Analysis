Data Analysis Internship Project using Python (Level 1 to Level 3)

# 🍽️ Cognifyz Data Analysis Internship

This project showcases my work during the Cognifyz Data Analysis Internship, where I analyzed real-world restaurant data using Python.

## 📊 Levels Covered

### 🔹 Level 1: Basic Analysis
- Top cuisines analysis  
- City-wise restaurant distribution  
- Price range distribution  
- Online delivery impact  

### 🔹 Level 2: Advanced Analysis
- Rating distribution  
- Cuisine combinations  
- Geographic analysis  
- Restaurant chains  

### 🔹 Level 3: Insight-Driven Analysis
- Review text analysis  
- Votes vs rating correlation  
- Price vs services analysis  

## 🔥 Key Insights
- Popular cuisines are not always highest rated  
- Votes have only a weak correlation with ratings  
- Restaurants are concentrated in specific regions  
- Premium restaurants offer more services  

## 🛠️ Tools Used
- Python  
- Pandas  
- Matplotlib  

## 📁 Project Structure
- Dataset → Raw data  
- Level1/2/3 → Analysis scripts  
- Outputs → Results & visualizations
